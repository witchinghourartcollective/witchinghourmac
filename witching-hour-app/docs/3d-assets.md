## How to run the app as-is

This is an Expo React Native app. From the repo:

```bash
cd /home/fletchervaughn/code/witching-hour/witching-hour-app
npm install
npm run start
```

Then:

- For web: press `w` in the Expo terminal, or run `npm run start:web`
- For iOS/Android: press `i` / `a` (requires simulators or devices)
