# Primary Sigil - Canonical Specification

## Geometry

- One vertical axis
- Four diagonal lines intersecting the center
- Symmetry is mandatory

## Constraints

- No curves
- No enclosure
- No rotation
- No skew

## Allowed variations

- Scale
- Color
- Opacity
- Stroke weight (uniform only)

## Prohibited

- Animation that implies motion
- Glow, blur, or shadow
- Decorative additions

## Reference

Rendered SVG lives at `public/brand/sigil-primary.svg`.
This file is the authoritative shape and meaning.
